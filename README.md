## Features

- **OOP Model**: Abstract `Pet` class; subclasses `Dog`, `Cat`, `Bird`.
- **Constructors**: Default and named constructors for pet creation.
- **Interface & Mixin**: `Friendly` interface; `Flyable` mixin for `Bird`.
- **Keywords**: Demonstrates `static`, `super`, and `this`.
- **UI**: List of 3 pet cards (Dog, Cat, Bird) and bottom text showing total pets.
